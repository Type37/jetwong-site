# jetwong.neocities.org — GitHub Mirror

Mirror of [jetwong.neocities.org](https://jetwong.neocities.org).  
Live at: **https://type37.github.io/jetwong-site/**

Synced from Neocities using `sync-to-github.sh`. Run the script whenever you want to pull the latest from Neocities into this repo.

## Sync

```bash
./sync-to-github.sh
# or with a custom commit message:
./sync-to-github.sh --message "Added Covert Ops generator"
```

Requires `curl`, `git`, `python3`.
